1480 // /// than a user defined contour curvature threshold value, set by default at 0.1. The threshold curvature
1481 // /// can also be defined as a multiple of the standard deviation of the curvature. Before this function is called

1487 // /// DEMs, Water Resources Research 49: 1-15
1488 // ///
1489 // /// @param window_radius Integer window radius - suggest 6m.
1490 // /// @param tan_curv_threshold Double curvature threshold value.
